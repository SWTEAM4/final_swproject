# Core Crypto Library (src/)

`src/`는 AES, SHA-512, HMAC-SHA512, PBKDF2-SHA512를 구현한 핵심 암호화 라이브러리 소스코드가 위치하는 디렉터리입니다.

본 라이브러리는 모듈형 구조로 설계되어 있으며, CLI, GUI, 테스트 프로그램 등 다양한 애플리케이션에서 공통으로 사용됩니다.

## 주요 기능

- **AES 암호화/복호화** (블록 암호 + CTR 모드 지원)
  - AES-128/192/256 지원
  - 스트리밍 방식 CTR 모드 구현

- **SHA-512 해시 함수**
  - 스트리밍 입력 처리 지원
  - 512비트(64바이트) 해시 출력

- **HMAC-SHA512**
  - 단일 호출 및 스트리밍 방식 지원
  - 메시지 무결성 검증

- **PBKDF2-SHA512 키 파생 함수**
  - 패스워드 기반 키 도출
  - 솔트 및 반복 횟수 지원

- **암호학적으로 안전한 난수 생성**
  - OpenSSL RAND_bytes 기반
  - Salt, Nonce 생성 지원

- **플랫폼 독립적 유틸리티**
  - Windows, macOS, Linux 지원
  - 파일 I/O 및 임시 파일 처리
