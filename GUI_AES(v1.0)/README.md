# GUI_AES (v1.0)

**GUI_AES**는 AES, SHA-512, HMAC을 구현한 암호화 라이브러리를 기반으로 한 **파일 암복호화 애플리케이션의 GUI(Graphical User Interface) 버전**입니다.  
Qt 프레임워크를 이용하여 직관적인 사용자 인터페이스에서 파일 암호화/복호화를 수행할 수 있습니다.

## 주요 기능

- **AES 암호화/복호화** (CTR 모드 지원)
- **SHA-512 해시 계산**
- **HMAC 인증**
- 직관적인 GUI 환경에서 파일 암복호화 수행
- 안전한 임시 파일 처리 및 스트리밍 방식 암복호화

⚠️**사용 전 반드시 라이브러리 소스코드 사용설명서를 읽어주시기를 바랍니다.**
