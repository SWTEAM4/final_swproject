# test_vectors

**test_vectors**는 AES-CTR, SHA-512, HMAC-SHA512, PBKDF2-SHA512 등 **암호화 라이브러리의 신뢰성과 정확성을 검증**하기 위한 테스트 코드와, CLI 애플리케이션의 **성능 테스트**를 포함하고 있습니다.  

이 폴더를 통해 사용자는 라이브러리가 올바르게 동작하는지 확인하고, 실제 애플리케이션 환경에서 성능을 평가할 수 있습니다.


## 주요 기능

- **AES-CTR 테스트 벡터 검증**
- **SHA-512 해시 테스트 벡터 검증**
- **HMAC-SHA512 테스트 벡터 검증**
- **PBKDF2-SHA512 키 파생 테스트**
- **CLI 애플리케이션 성능 테스트**
