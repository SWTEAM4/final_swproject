# 2025-2 보안 SW 프로그래밍: 파일 암복호화 라이브러리 및 애플리케이션

본 프로젝트는 **2025-2 보안SW프로그래밍 수업**에서 구현한 **AES, SHA-512, HMAC 기반 암호화 라이브러리**와, 이를 이용한 **파일 암복호화 애플리케이션**을 포함하고 있습니다.  
프로젝트는 CLI와 GUI 버전으로 제공되며, 테스트 벡터와 성능 검증 코드도 포함되어 있어 안전성과 신뢰성을 확인할 수 있습니다.


## 주요 기능

- **암호화 라이브러리 (src)**
  - AES-CTR 암호화/복호화
  - SHA-512 해시
  - HMAC-SHA512
  - PBKDF2-SHA512 키 파생

- **CLI_AES**
  - CLI 환경에서 파일 암호화/복호화
  - SHA-512 해시 계산
  - HMAC 검증

- **GUI_AES**
  - Qt 기반 GUI 환경에서 파일 암호화/복호화
  - SHA-512 해시 및 HMAC 검증
  - 직관적인 인터페이스 제공

- **test_vectors**
  - 암호화 알고리즘 테스트 벡터 검증 (AES-CTR, SHA-512, HMAC-SHA512, PBKDF2-SHA512)
  - CLI 애플리케이션 성능 테스트

- **solution**
  - GUI를 제외한 각 암호 모듈 및 애플리케이션 소스코드에 대한 참고용 솔루션 구현을 포함
